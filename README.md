## Alcance del Proyecto
El presente proyecto tiene como objetivo realizar la certificación técnica de enlaces de par trenzado (UTP) dentro de un entorno de laboratorio, simulando una implementación de red LAN corporativa. 

La actividad se desglosa en los siguientes puntos clave:

1. *Validación Normativa:* Se asegura que los puntos de red cumplan con los estándares internacionales *TIA/EIA-568* para transmisiones de alta velocidad.
2. *Evaluación de Desempeño:* Se miden y analizan parámetros eléctricos críticos como la *Atenuación, el **NEXT* y la relación *SNR*, garantizando un ancho de banda de hasta 1 Gbps (1000Base-T).
3. *Certificación de Campo:* Se utiliza equipamiento profesional (*NetChaser TNC950) para ejecutar pruebas de **BERT* (tasa de error de bits) y *TDR* (localización de fallas por reflexión).
4. *Análisis de Integridad:* Se documentan y justifican técnicamente los resultados de "Pasa/Falla" obtenidos, identificando problemas comunes como fallas en el mapa de cableado (Wiremap) o excesos de longitud.
